🔗 位置关系：第三类小品词空间对比

ABOVE / BELOW / BEFORE / BEHIND / BETWEEN / AMONG / BESIDE / BENEATH / OPPOSITE / BEYOND

🎯 总场景：两个物体在不同方向的相对位置
⬆️⬇️ 上下关系
ABOVE
在……上方（不接触）。高于但不一定垂直。和OVER区别：OVER强调越过/跨越，ABOVE只强调位置更高。
above the clouds / above average / above all
BELOW
在……下方（不接触）。低于。和UNDER区别：UNDER强调垂直下方，BELOW只是"更低"。
below zero / below average / see below
↔️ 前后关系
BEFORE
在……前面。也用于时间"在……之前"。
before the door / before lunch / come before / before long
BEHIND
在……后面。抽象：落后、支持、隐秘原因。
behind the door / fall behind / the reason behind
🧩 中间 / 旁边 / 对面 / 更远处
BETWEEN
在两者之间。只在两个事物中间。抽象：选择、关系。
between you and me / choose between / come between
AMONG
在多个之间。三个或以上事物中间。
among friends / among the crowd / among others
BESIDE
在……旁边。紧挨着。BY也表"旁边"，BESIDE更强调紧邻。
beside the bed / beside the point / beside yourself
BENEATH
在……正下方。接触或不接触。抽象：低于（地位/尊严）。
beneath the surface / beneath you / beneath contempt
OPPOSITE
在对面。正对面。抽象：相反的、对立的。
opposite the store / opposite direction / opposite sex
BEYOND
在更远处 / 超出。超过界限/范围。抽象：超出能力。
beyond the hill / beyond belief / beyond repair / beyond me
