# 英语常用副词参考列表

按语义类别分类。**加粗**为特别适合程度光谱展示的词。

## 频率类 Frequency (阶梯递减)

**ALWAYS / USUALLY / OFTEN / SOMETIMES / RARELY / NEVER**
**CONSTANTLY**, **FREQUENTLY**, **REGULARLY**, OCCASIONALLY, SELDOM, HARDLY EVER

→ 适合用时间线 + 覆盖密度光谱展示

## 程度类 Degree

**VERY**, **QUITE**, **RATHER**, **PRETTY**, **FAIRLY**, **SOMEWHAT**, **SLIGHTLY**, **BARELY**
**COMPLETELY / PARTIALLY**, **TOTALLY**, **ABSOLUTELY**, **ENTIRELY**, **ALMOST**, **NEARLY**
**HIGHLY**, **DEEPLY**, **BADLY**, **SERIOUSLY**, EXTREMELY, INTENSELY, MILDLY

→ 适合用进度条/刻度尺光谱展示

## 速度类 Speed

**QUICKLY / SLOWLY**, **RAPIDLY**, **FAST**, **GRADUALLY**, **SUDDENLY**, **IMMEDIATELY**, **PROMPTLY**, STEADILY

## 方式类 Manner

**CAREFULLY / CARELESSLY**, **LOUDLY / QUIETLY**, **EASILY**, **HARDLY**, SMOOTHLY, SUDDENLY, GENTLY, ROUGHLY, PROPERLY, NEATLY

## 时间类 Time

**NOW / THEN**, **SOON**, **LATER / EARLIER**, **RECENTLY**, **PREVIOUSLY**, **IMMEDIATELY**, FINALLY, EVENTUALLY, CURRENTLY, PRESENTLY

## 地点类 Place

**HERE / THERE**, **INSIDE / OUTSIDE**, **UPSTAIRS / DOWNSTAIRS**, **EVERYWHERE / NOWHERE**, ABOVE / BELOW, AWAY, BACK, FORWARD

## 逻辑/态度类 Logic & Attitude

**ALSO**, **TOO**, **AS WELL**, **ONLY**, **JUST**, **EVEN**, **INDEED**, **ACTUALLY**, **REALLY**, **ABSOLUTELY**, **DEFINITELY**, PROBABLY, POSSIBLY, PERHAPS, MAYBE, CERTAINLY, SURELY, OBVIOUSLY, ESSENTIALLY, BASICALLY

## 连接类 Conjunctive Adverbs

**HOWEVER**, **THEREFORE**, **MOREOVER**, **FURTHERMORE**, **NEVERTHELESS**, **OTHERWISE**, **MEANWHILE**, **BESIDES**, **INSTEAD**, **OTHERWISE**

## 适合光谱递进展示的词组

| 光谱 | 阶段 |
|------|------|
| NEVER → RARELY → SOMETIMES → OFTEN → ALWAYS | 频率从低到高 |
| BARELY → SLIGHTLY → SOMEWHAT → QUITE → VERY → COMPLETELY | 程度从浅到深 |
| SLOWLY → STEADILY → QUICKLY → RAPIDLY | 速度从慢到快 |
| QUIETLY → LOUDLY → DEAFENINGLY | 音量从小到大 |
| NEVER → HARDLY EVER → SOMETIMES → USUALLY → ALWAYS | 频率光谱（细化） |
| PARTIALLY → ALMOST → NEARLY → COMPLETELY | 完成度从低到高 |
