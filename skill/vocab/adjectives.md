# 英语常用形容词参考列表

按"是否适合物理→抽象图解"分类。**加粗**为特别适合视觉化展示的词（有物理意象 / 有程度渐变 / 可画光谱）。

## 空间/形状类 Spatial & Shape

**BIG / SMALL**, **LARGE / TINY**, **LONG / SHORT**, **WIDE / NARROW**, **HIGH / LOW**, **DEEP / SHALLOW**, **THICK / THIN**, **FAT / THIN**, **FLAT**, **ROUND**, **SHARP / DULL**, **SMOOTH / ROUGH**, **HARD / SOFT**, **HEAVY / LIGHT**, **TALL / SHORT**, **WISPY**

## 松紧/力度类 Tension & Force

**TIGHT / LOOSE**, **SLACK**, **TAUT**, **TENSE / RELAXED**, **STIFF / LIMP**, **FIRM**, **SOLID**, **BRITTLE**, **FLAKY**, **CRUMBLY**, **STRETCHY**, **ELASTIC**, **RIGID / FLEXIBLE**, **STUCK**, **SNAPPED**

## 温度/触觉类 Temperature & Touch

**HOT / COLD**, **WARM / COOL**, **FREEZING / BOILING**, **MILD**, **WET / DRY**, **DAMP**, **MOIST**, **ICY**, **BURNING**

## 光亮/颜色类 Light & Color

**BRIGHT / DARK**, **LIGHT / HEAVY** (光照/重量两个义项), **CLEAR / CLOUDY**, **DULL / SHINY**, **PALE / VIVID**, **TRANSPARENT / OPAQUE**

## 速度/频率类 Speed & Frequency

**FAST / SLOW**, **QUICK**, **RAPID**, **SUDDEN**, **GRADUAL**, **CONSTANT**, **STEADY**, **RARE / COMMON**, **FREQUENT**

## 程度/强度类 Degree & Intensity

**FULL / EMPTY**, **COMPLETE / PARTIAL**, **TOTAL**, **ABSOLUTE**, **EXTREME / MODERATE / MILD**, **STRONG / WEAK**, **POWERFUL**, **INTENSE**, **SEVERE**, **SLIGHT**

## 干净/状态类 Cleanliness & Condition

**CLEAN / DIRTY**, **PURE / IMPURE**, **FRESH / STALE**, **NEW / OLD**, **YOUNG / AGED**, **RAW / RIPE**, **FRESH / ROTTEN**, **WHOLE / BROKEN**, **DAMAGED**, **PERFECT / FLAWED**, **DONE**, **GONE**, **WORN (OUT)**, **REFINED**, **SPOILT / SPOILED**

## 情绪/性格类 Emotion & Character

**HAPPY / SAD**, **ANGRY / CALM**, **NERVOUS / RELAXED**, **PROUD / ASHAMED**, **BRAVE / COWARDLY**, **KIND / CRUEL**, **POLITE / RUDE**, **HONEST / DISHONEST**, **LOYAL / DISLOYAL**, **GENEROUS / STINGY**, **SERENE**, **FLUSTERED**, **LOWKEY**

（注意：情绪类形容词多为抽象概念，图解时需用人物表情/肢体动作作为物理载体）

## 适合光谱递进展示的词组

| 光谱 | 阶段 |
|------|------|
| TIGHT → SLACK → LOOSE → BROKEN | 绳子从绷紧到断裂 |
| FULL → HALF → EMPTY | 容器从满到空 |
| CLEAN → DIRTY → FILTHY | 物体从干净到肮脏 |
| COLD → COOL → WARM → HOT → BOILING | 温度从低到高 |
| DARK → DIM → BRIGHT → GLARING | 光线从暗到亮 |
| DRY → DAMP → WET → SOAKED | 湿度从干到透 |
| HARD → FIRM → SOFT → MUSHY | 硬度从硬到软 |
| SLOW → MODERATE → FAST → RAPID | 速度从慢到快 |
| RARE → UNCOMMON → COMMON → FREQUENT | 频率从低到高 |
| SHARP → DULL → BLUNT | 锋利度递减 |
