## 适合光谱递进展示的词组

| 光谱 | 阶段 |
|------|------|
| DRIZZLE → RAIN → DOWNPOUR | 降水量从少到多 |
| BREEZE → WIND → GALE → STORM | 风力从弱到强 |
| WHISPER → VOICE → SHOUT | 声音从小到大 |
| DRIP → FLOW → STREAM → POUR | 水流从慢到快 |
| SPARK → FLAME → FIRE → INFERNO | 火势从小到大 |
| DUST → DIRT → MUD | 脏污程度 |
| PUDDLE → POOL → POND → LAKE | 水体大小 |

## 常见隐喻转化名词

- **HEART** → 核心 (the heart of the issue)
- **BACK** → 支撑/后背 (behind your back, back support)
- **FOOT** → 底部 (foot of the mountain, foot of the page)
- **HEAD** → 顶部/领导 (head of the table, head of the company)
- **BRIDGE** → 连接 (build bridges between cultures)
- **WALL** → 障碍 (hit a wall, wall of silence)
- **ROAD / PATH** → 途径 (career path, road to success)
- **WEIGHT** → 负担/重要性 (weight of responsibility, carry weight)
- **LIGHT** → 理解/启示 (shed light on, see the light)
- **SHADOW** → 负面影响 (cast a shadow over)
