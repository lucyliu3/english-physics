核心意象对比

CHECK /tʃek/

verb · noun · 检查 / 核对 / 验证
本义 逐一核对清单上的项目，确认无误后打勾 ✓
引申 任何形式的验证、确认、检查动作 "Please check your answers before submitting." 提交前请检查答案。
💡 理解关键：CHECK 的物理画面是"手指着清单上的项目，逐个比对确认"。打勾 ✓ 是确认动作的视觉符号。所有抽象用法（check in/out/off/on）都源于"与标准清单比对"这个核心动作。

CHECK vs TICK /tʃek/ vs /tɪk/

英美用法差异
美式英语 ✓ check the box, check your answers, check in/out
英式英语 ✔ tick the box, tick off the list "Check/Tick the correct answer." 勾选正确答案。
💡 关键区别：美式 check 用法范围远超英式 tick。Check 可以表示"检查、验证、寄存、下棋将军、支票"等（因为"打勾确认" → "验证" → "确认支付" → 支票）。Tick 基本只表示"打勾"这个动作本身。

🔍 重点短语详解
check in
办理入住 / 签到
🧠 演变：在酒店前台，你的名字在"客人清单"上 → 前台找到你的名字，标记"已到"（打勾）→ 你被登记进入系统。所以 check in = 登记+进入。
"We checked in at 3 PM." 我们下午3点办理了入住。
check out
退房 / 结账离开 / 查看
🧠 演变：退房时前台在你的名字旁标记"已离开"（check out = 从清单上勾掉）→ 引申为"结账离开"。口语中 check out (something) = 去看看/瞧瞧（像在清单上"查看"某个项目）。
"Check out that new cafe!" 去瞧瞧那家新咖啡馆！
check off
勾掉 / 标记完成
🧠 演变：清单上的项目完成 → 在项目旁打勾 ✓，表示"已从待办事项中移除"。off 表示"脱离清单"，即"已完成不再操心"。
"Check off each item as you finish." 每完成一项就把它勾掉。
check up on
查看/探望（确认状态正常）
🧠 演变：清单上某人的状态 → 需要去核实他是否安好 → up 表示"接近查看细节"，on 表示"关于某人"。
"My mom always checks up on me." 我妈总是来看我好不好。
check on
确认状态 / 检查进展
🧠 演变：on 表示"针对某个对象"。从清单上某个项目 → 去确认它的当前状态。比 check up on 更中性（不一定是关心，可能是监督）。
"I'll check on the dinner." 我去看看晚饭怎么样了。
check yourself
克制自己 / 反省言行
🧠 演变：把"自己"当成清单上的项目 → 审视自己的言行是否符合标准 → 在失控前"检查自己"。check 从"检查清单"变为"控制/约束"。
"You need to check yourself before you speak." 说话之前你得克制一下自己。
check the bill
核对账单
🧠 演变：账单上的每一项费用 → 逐项比对（点了什么菜、价格是否合理）→ 确认无误后付钱。这是"check = 核对"最经典的商业场景。
"Always check the bill before paying." 付钱前一定要核对账单。
check out (look at)
看看/瞧瞧（口语）
🧠 演变：从图书馆 checkout（借出登记）→ 引申为"把某物从橱窗/陈列中拿出来看看"。现代口语 = "快去瞧瞧！"
"Check out this video!" 快看这个视频！
易混对比
CHECK vs VERIFY
快速核对 vs 精确验证
💡 区别：check 是"看一下有没有问题"，可能只是扫一眼；verify 是"用标准数据逐字比对"，要求100%匹配。你要"check 有没有错别字"但要"verify 银行卡号"。
CONFIRM vs INSPECT
确认 vs 检视
💡 区别：confirm 只是获取"是/否"的确认（confirm a reservation）；inspect 是要用放大镜仔细看每一个细节来找问题（inspect a building）。Check 介于两者之间——可以快速 confirm 也可以仔细 inspect。

反义 / 相似词
核心词

CHECK — 检查/核对 通用，范围最广

TICK — 打勾 英式英语，基本只用于勾选

近义辨析

VERIFY — 精确验证 逐条比对

CONFIRM — 确认 二次确认是否如此

INSPECT — 检视 仔细检查找问题

反义/易混

IGNORE — 忽略 跳过不检查

OVERLOOK — 看漏 检查了但没注意到

CHECKMATE — 将军 国际象棋"将死"（check的同源词）
