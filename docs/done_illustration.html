核心意象对比

DONE /dʌn/

ADJECTIVE · 完成；已结束；煮熟
本义 DO 的过去分词，"已完成" —— 一个任务/动作从进行状态转变为完成状态
核心意象 清单上的项目被勾选 ✔️，从"待办"转变为"办结" "The report is done. You can review it now." 报告已完成，你可以审阅了。
💡 理解关键：DONE 的核心画面是"从待办状态转化为已完成状态"——像任务清单上打勾一样，一个状态翻转的瞬间。这种"翻转"后来被用于各种抽象场景：关系结束（done with someone）、食物煮熟（meat is done）、命运已定（done for）。

DONE = 已结束 / 已烹饪

状态形容词 · 物理状态 + 抽象状态
延伸用法 从"任务完成"延伸到"食物煮熟"、"关系结束"、"命运注定" "Is the steak done? Medium rare, please." (牛排好了吗？五分熟，谢谢。) "We're done. Don't call me again." (我们结束了。别再打给我。)
🧠 如何演变的：任务清单打勾 → 任何达到终点状态 = done。食物从生到熟是"一个过程完成"，关系从存在到不存在是"一个状态的终结"，命运从悬而未决到已定是"过程不可逆地结束"。核心始终是：从进行中/未完成，到完成/终结的状态转换。
程度光谱：从 NOT DONE 到 DONE
💡 光谱含义：DONE 不是一个"有或无"的二元状态，而是一个进度光谱的终点。从 NOT DONE（红色/未开始）到 PENDING（黄色/排队）到 IN PROGRESS（橙色/执行中）到 ALMOST DONE（浅绿/收尾）再到 DONE（深绿/完成）。这个光谱解释了为什么 "almost done" 和 "nearly done" 是合理的表达 —— 完成是一个渐变过程。

🧠 演化逻辑：DONE 的核心 = "从进行态到完成态的翻转"。这个"翻转"应用到不同领域：
- 任务上 = 作业做完了 ✅
- 烹饪上 = 肉烤熟了 🥩
- 关系上 = 和某人掰了（be done with sb）💔
- 命运上 = 完蛋了（done for）= "你的过程已经走完，结果定了" ☠️
- 交易上 = 成交了（done deal）= "事情已经定下来，不再是讨论阶段" 🤝
- 精力上 = 累垮了（done in）= "能量被消耗完了" 😵
所有用法共享同一个底层意象：状态从"未完成"翻转到了"已完成"。
易混对比
DONE vs FINISHED vs COMPLETED
DONE 最口语化，侧重"结束/终止"状态；FINISHED 侧重"所有步骤完成"；COMPLETED 正式书面用语，侧重"满足要求"。
"I'm done with this job" = 我辞职了（不想干了）；"I finished this job" = 我干完了（任务完成了）。
with = 和某事/某人的关系主动终止 | done for = 被动下场/完蛋了 | done in = 精力被消耗殆尽 小介词 with/for/in 彻底改变了 done 的语义方向 主动结束 被动毁灭 能量耗尽
DONE WITH vs DONE FOR vs DONE IN
介词的改变决定了 DONE 的语义色彩。WITH = 主动结束一段关系或活动（中性）；FOR = "为了...而完了"（负面、毁灭性）；IN = 能量被消耗殆尽（疲惫）。
常见搭配 / 短语

📖 核心短语搭配

be done with (something/someone) 和...结束 / 不再与...有关系  |  📖 主动终止与某人或某事的关系
be done for 完蛋了 / 死定了  |  📖 无法挽回的糟糕结局
be done in 累瘫了 / 精疲力尽  |  📖 精力被完全消耗
over and done with 彻底结束 / 翻篇了  |  📖 强调"过去了不再想"
well done 干得好! / 全熟牛排  |  📖 称赞完成得好；牛排烹饪程度
easier said than done 说来容易做来难  |  📖 固定谚语，表示知易行难
done deal 既成事实 / 拍板定案  |  📖 交易或决定已完成，不可改变
when all is said and done 归根结底 / 说到底  |  📖 总结性表达，表示"考虑所有因素之后"
no sooner said than done 说做就做 / 立即执行  |  📖 表示行动极其迅速
what's done is done 木已成舟 / 覆水难收  |  📖 劝人接受已发生的事实
have done with it 干脆做完 / 一了百了  |  📖 鼓励快速完成以免拖延
hard done by 受到不公平对待  |  📖 英式英语，感觉被亏待

📖 日常场景用法

The work is done. 工作做完了  |  📖 最基本的用法，表示任务完成
Is the meat done? 肉熟了吗？  |  📖 烹饪场景，询问食物是否达到可食状态
I'm done with this job. 我不干了 / 我辞职  |  📖 表示决定离开或结束某项工作
We're done here. 我们这里结束了  |  📖 表示会议/谈话/活动结束
You're done. 你完了 / 你搞定了  |  📖 根据语境可以是正面"你完成了"或负面"你完蛋了"
Done and dusted. 彻底完成 / 尘埃落定  |  📖 英式俚语，事情完全结束
That's a done deal. 这事定了 / 板上钉钉  |  📖 交易或协议已不可更改
Done! 搞定！ / 成交！  |  📖 单字成句，表示"没问题/说好了"
💡 be done with 的演变逻辑：任务清单打勾 = "不再和这任务有关系" → 引申到人和人的关系："I'm done with him" = "我和他的关系已经打勾了/终结了"。
💡 be done for 的演变逻辑："for" 表示目的/方向。done for = "为了...而完成" → "你的命运已经朝着某个注定方向走完了" = 完蛋了。
💡 be done in 的演变逻辑："in" 表示进入内部。done in = "从内部被做完了" = 你的精力/能量从内部被掏空 = 累垮了。
💡 done deal 的演变逻辑：交易从"待商议"状态 → 打勾 = 成交。deal 被 done 了 = "交易已完成"，强调不可撤销。
💡 what's done is done / over and done with 的演变逻辑：重复 done = 强调状态已经不可逆，多次打勾确认。over = 越过，done = 完成，= 已经过去了并且完成了，强调"翻篇"。
反义 / 相似词
🔴 反义词 (Antonyms)

UNDONE — 未完成的 "left undone" = 搁置未做

INCOMPLETE — 不完整的 缺了某些部分

PENDING — 待处理的 还在等待队列中

UNFINISHED — 未结束的 没有走完所有步骤

UNRESOLVED — 未解决的 问题还没找到答案

🔵 近义词辨析 (Synonyms)

FINISHED — 侧重"走完所有步骤" 比 done 更强调过程完整性

COMPLETED — 正式"完美收官" 满足所有标准/要求

OVER — 侧重"时间上结束了" "The movie is over."

THROUGH — 侧重"从开始到结束穿越了" "I'm through with this."

SETTLED — 侧重"已经被解决" 纠纷/争议/账目

🟣 易混词提醒

DONE vs DID — done 是过去分词（形容词/完成态），did 是过去式（动作态）
"I did it."(我做了这个动作) vs "It is done."(这件事完成了)

DONE vs DO — done 强调结果状态，do 强调动作过程
"What do you do?" vs "What is done?"

DONE vs DOING — done = 已完成，doing = 进行中
完全的时态对立
