🧭 方向 / 路径：第二类小品词空间对比

UP / DOWN / TO / FROM / INTO / OUT OF / TOWARD / THROUGH / ACROSS / OVER / UNDER

🎯 总场景：一条路 + 一个盒子 + 一个障碍物
⬆️⬇️ 上下方向
UP
从低到高。向上移动。抽象：增加、改善、出现。
Get up / Show up / Give up / Grow up / Up to you
DOWN
从高到低。向下移动。抽象：减少、变差、崩溃。
Sit down / Break down / Calm down / Write down
🎯 到达 / 来源
TO
朝向 / 到达。指向目的地，强调"到达"。
Come to / Go to / Lead to / Stick to / to be honest
FROM
从某处来 / 来源。运动的起点。
Come from / Suffer from / Tell from / from now on
🚪 进入 / 出来
INTO
从外到内。强调进入的过程（IN只强调在里面）。
Get into / Run into / Look into / into details
OUT OF
从内到外。出来的过程。抽象：缺乏/离开状态。
Get out of / out of stock / out of breath
🚶 方向：朝向 / 穿过 / 横穿 / 越过 / 下方
TOWARD(S)
朝向（不一定到达）。往那个方向走。
Go toward / attitude toward / work toward a goal
THROUGH
从一端穿到另一端。贯穿过程。
Go through / Get through / Break through
ACROSS
横穿一个平面。从平面一边到另一边。
Come across / Get across / across the street
OVER
从上方越过 / 覆盖。跨越障碍或覆盖表面。
Come over / Get over / Take over / Think over / over there
UNDER
在……下面 / 低于。正下方。抽象：受控、进行中。
under the table / under control / under construction
