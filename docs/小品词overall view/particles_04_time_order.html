⏱️ 时间 / 顺序：第四类小品词空间对比

AFTER / BEFORE / AFTERWARD(S) / FORWARD / BACKWARD(S) / SINCE / UNTIL

注意：时间词的本质就是把空间关系映射到时间轴上。看到任何一个时间词，先想它的空间意象。

🎯 时间轴上的空间关系
⏮️⏭️ 在……之前 / 之后（顺序关系）
BEFORE
在……之前。从空间"在前面"映射到时间"在之前"。和AFTER相反。
before lunch / before the war / before class / before long
AFTER
在……之后。从空间"跟在后面"映射到时间"晚于"。抽象：追求、模仿。
after school / go after / take after / after all
⏪⏩ 向前 / 向后（方向）
FORWARD
向前。朝前方移动。抽象：进步、提出、期待。
go forward / come forward / put forward / look forward to
BACKWARD(S)
向后。朝后方移动。抽象：倒退、退步。
step backward / bend over backward / backward thinking
🔄 AFTERWARD(S) 和 -WARD 系列
AFTERWARD(S)
之后 / 后来= AFTER（之后）+ WARD（朝……方向）。副词，表示"在那之后的时间点上"。
soon afterward / shortly afterward / What happened afterward?
-WARD 系列总览
AFTER + WARD = 之后 / FOR + WARD = 向前 / BACK + WARD = 向后 / TO + WARD = 朝 / UP + WARD = 向上 / DOWN + WARD = 向下 / OUT + WARD = 向外 / IN + WARD = 向内。全部来自空间方向！
same pattern: afterward, forward, backward, toward, upward, downward, outward, inward
📌 时间端点：SINCE / UNTIL
SINCE
自从。从过去某个时间点一直到现在。强调"从那时起"。
since 2020 / since then / since I was a kid / ever since
UNTIL
直到。从现在到未来的某个时间点。强调"到那时为止"。
until tomorrow / until now / until I come back / not until
