⬅️ PULL

朝自己方向用力 — 从拉绳子到拉票到拉倒

核心意象对比

PULL /pʊl/

verb · 拉 / 拖 / 拽
本义 朝自己方向用力，使物体靠近
引申义 1 拉（门、绳子、抽屉）
引申义 2 抽象：拉拢、吸引、抽出 "pull the door" 拉门
"pull an all-nighter" 熬通宵
"pull the trigger" 扣扳机
💡 核心逻辑：PULL 的本质是"施加向自己的力"——所以引申出"吸引"（拉近注意力）、"拉出"（从内部抽出来）、"拉拢"（把人拉到这边）。所有用法都包含一个方向：朝施力者靠近。

PULL vs PUSH

最经典的一对反义动词
PULL = 朝自己方向用力（物体靠近）
PUSH = 朝远离自己的方向用力（物体远离） "pull the door open" 拉开门
"push the door open" 推开门
🧠 如何演变的：物理方向决定了抽象方向——pull 引申为"吸引"（朝自己拉近注意力），push 引申为"推动/推进"（把事物往前送）。"pull someone aside"（拉到一边私下说）vs "push someone away"（推开一个人），物理方向完全对应抽象含义的方向。

🧠 核心逻辑链：拉物体（物理）→ 从内部"拉"出来 = pull out/up（抽出）→ 🚗 把车"拉"到某处停下 = pull up / 从车道"拉"到路边 = pull over / "拉"进车位 = pull into / "拉"出去 = pull out / "拉"离 = pull away（驾车）→ 把注意力"拉"过来 = attract/pull a crowd（吸引）→ 把人"拉"到自己这边 = pull strings（拉关系）→ 把表情/动作"拉"出来 = pull a face（做鬼脸）→ 把任务"拉"到终点 = pull off / pull through（完成）。
所有用法共享一个画面：用外力把目标物朝施力方向移动。
🚗 驾驶场景 — 美语口语极高频
🧠 核心逻辑：所有这些驾驶用法都来源于同一个物理意象——"对车的操控力朝某个方向"。pull up = 把马/车"拉到"某处停下（源于勒马缰绳）；pull over = 把车"拉过"到路边；pull into = "拉进"某个空间；pull out of = 从某处"拉出来"；pull away = "拉离"当前位置。
💡 美语驾车场景的 pull 用法极其高频，native speaker 几乎每天都会用到。英国人更习惯用 pull in 表示靠边，美国人用 pull over。
🚗 pull up (to) = 开车到某地停下。物理来源：骑马时拉缰绳（pull up on the reins）让马停下 → 引申为车"到达+停下"。
"Pull up to the grocery store, I'll be right out."（开到杂货店门口，我马上出来）
"Your Uber just pulled up."（你 Uber 到了）
💡 对比：pull up 强调"到达并停靠"，而 pull over 强调"从车道移到路边"。
🚗 pull over = 靠边停车。物理画面：把车"拉过来"到路边。最常见的场景是被警察拦停——"I got pulled over."
"The cop pulled me over for speeding."（警察因为我超速拦了我）
"Pull over! I'm gonna be sick."（靠边停！我要吐了）
💡 注意：pull over 一定是从车道移到路边，而不是开进停车场。
🚗 pull into = 开进（一个封闭/特定空间）。物理画面：把"拉"这个动作的方向指向某个空间内部。
"She pulled into the driveway/parking lot."（她开进了车道/停车场）
"Pull into that spot on the right."（停进右边那个车位）
💡 反义：pull out of — 从停车位/车道驶出。
🚗 pull out / pull out of = 驶出（某空间）进入车道。物理画面：把车从空间里"拉出去"。
"Wait until it's safe to pull out."（等安全了再开出去）
"He pulled out of the parking lot without looking."（他没看就开出了停车场）
💡 pull out of 也可以抽象用：pull out of a deal（退出交易）。
🚗 pull away = 从停止/路边状态开始行驶离开。强调"驶离"这个动作。
"The bus pulled away just as I reached the stop."（我刚到站公交车就开走了）
"Pull away from the curb slowly."（从路边慢慢驶离）
🚗 pull a U-turn = 掉头。口语化表达，比 make a U-turn 更随意。物理画面：把车"拉"出一个U形路线。
"I missed the exit, so I had to pull a U-turn."（我错过了出口，只好掉头）
💡 类似扩展：pull a three-point turn（做三点掉头），但不如 make/do 常见。
🚗 pull off (the road/highway) = 驶离主路/高速公路出口。物理画面：把车"拉"离主干道。
"We pulled off the highway to grab some lunch."（我们开下高速公路去吃午饭）
"Pull off at Exit 23."（在23号出口下去）
💡 对比：pull over（靠边停）vs pull off（驶离主干道继续开或停）。（靠边停）和 pull off（驶离主干道继续开或停）。
🚗 pull out in front of someone = 插到别人前面（危险驾驶）。物理画面：从路边/停车场"拉出去"正好到了别人前面。
"Some idiot pulled out in front of me."（有个傻X突然插到我前面）
"You gotta check your blind spot before you pull out."（驶出前要看盲点）
💡 这是美语驾车场景中高频抱怨用语。
易混对比
PULL vs DRAG vs TOW — 拉的方式不同
PULL = 一般性地拉；DRAG = 费力地拖（有摩擦阻力，沿地面）；TOW = 用绳索牵引（车辆/船只）。
PULL ON vs PULL AT — 用力方式不同
PULL ON 强调持续用力（常用于穿衣服: pull on a sweater），也可表示"拉绳子"；PULL AT 强调轻/反复拉扯（pull at one's sleeve 扯某人袖子）。

pull the plug 拔插头  |  📖 把插头拉出来 = 停止供电，引申为终止项目
pull a muscle / hamstring 拉伤肌肉  |  📖 肌肉被过度"拉"伸受伤
pull a tooth / cork / weed 拔/拽出来  |  📖 从固定位置拉出

🗣️ 口语高频 — 抽象 PULL

pull a crowd / audience 吸引人群/观众  |  📖 把人的注意力"拉"过来
pull the trigger on sth. 下决心做某事  |  📖 扣扳机=做出关键决定
pull an all-nighter 熬通宵  |  📖 "拉"出一整晚不睡
pull strings 走后门/拉关系  |  📖 像操控木偶一样"拉"背后的线
pull a face 做鬼脸  |  📖 把表情"拉"出来
pull a stunt / trick 搞花样/耍花招  |  📖 "拉"出某个动作来
pull through 挺过来/渡过难关  |  📖 把自己"拉"过难关
pull off 成功做到（困难的事）  |  📖 把不可能的事"拉"到终点
pull out of a deal 退出交易  |  📖 把自己"拉"出去
pull yourself together 振作起来/冷静  |  📖 把散落的自己"拉"回一体
pull for sb. / a team 支持/为某人加油  |  📖 心"拉"向某个方向=倾向支持
🔄 pull yourself together = 振作/冷静下来。物理画面：你的状态"散了"（像一条松掉的绳子），你用力把自己"拉回"一个完整集中的状态。美语口语极高频，情绪崩溃时别人会对你说 "Pull yourself together!"。
🔄 pull for (someone) = 支持/为某人加油。物理画面：在心里"把支持和鼓励拉向"某人。"root for"的同义表达。体育比赛、生病康复、困难时期很常用。
"We're all pulling for you to get well soon."（我们都支持你快快好起来）
"I'm pulling for the underdog."（我支持弱势一方）
🔄 pull someone's leg = 开某人玩笑/逗某人。来源说法不一，但最形象的理解：你"拉"某人的腿让他踉跄 = 你"戏弄"他。不是恶意的，是朋友间逗着玩。"I'm just pulling your leg!" = "我逗你玩的！"
🔄 pull rank = 利用职权压人/摆官威。rank = 军衔/级别。物理画面：你把"军衔"这张牌"拉"出来 = 拿级别来压人。"Don't pull rank on me!" = "别拿级别压我！"

🚫 PULL 的反义词

🔗 近义辨析

PULL vs DRAG：一般拉 vs 费力拖

PULL vs TOW：人的拉 vs 车辆牵引

PULL vs TUG：持续拉 vs 猛拉/拽一下

PULL vs YANK：正常拉 vs 猛地一扯
