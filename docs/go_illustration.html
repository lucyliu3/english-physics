GO 的空间意象

核心逻辑：从某处向目标移动 → 最基础的"位移"动词

核心空间逻辑
GO
从起点向目的地移动。
核心 = 位移（强调"离开现在位置，前往别处"）
GO + 小品词
不同的方向小品词 = 不同的移动轨迹。
核心 = 方向（小品词决定了位移的具体路径）
四大自然类别

基本方向 go + 方向小品词

go in / go out / go up / go down / go through / go across
用 GO 来表达"往某个方向移动"，方向由小品词决定。空间位移。 "Go in." / "Go out." / "Go up the stairs."

前往 / 离开 go to / go from

go to work / go home / go to bed / go away / go off
GO 的核心机制：从A到B。to 标记目的地，from/away 标记离开。 "I'm going to the store." / "Go away!"

去做某事 go + V-ing / go on

go shopping / go swimming / go on a trip / go on vacation
GO + 动名词 = "去做某活动"。从"不动"到"在做"的状态切换。活动位移。 "Let's go swimming." / "go on a date"

抽象推进 go on / go ahead / go by

go on / go ahead / go by / go wrong / go well
从空间位移延伸到时间 / 状态 / 事件的推进。事情"向前走"就叫 go。 "Go ahead." / "How's it going?" / "What's going on?"
GO 高频用法
① 基本位移 — go + 小品词
Please go in. 请进去。
Don't go out in the rain. 别冒雨出去。
Go up the stairs. 上楼。
Go down the street. 沿着街走下去。
Go through the tunnel. 穿过隧道。
Go across the bridge. 走过那座桥。
The road goes around the mountain. 这条路绕着山走。
He went past me without saying hi. 他从我旁边走过去没打招呼。
② go to / go away / go off — 目的地 vs 离开（超高頻日常）
I go to work at 8. 我8点上班（去公司）。
Go to bed! 去睡觉！（对孩子说）
She went to the doctor. 她去看医生了。
Go away! 走开！
The alarm went off. 闹钟响了 / 警报响了。
He went off to find a taxi. 他走开去找出租车了。
Go home. 回家。（注意！home前没有to）
Let's go that way. 咱们走那边。
③ go + V-ing / go on —— 去做某活动（超高頻口语）
Let's go swimming. 咱们去游泳吧。
I want to go shopping. 我想去逛街。
They go camping every summer. 他们每年夏天去露营。
She went on a trip to Japan. 她去日本旅行了。
We're going on vacation. 我们要去度假了。
Go on a date. 去约会。
He went on a diet. 他开始节食了。
They went on a tour. 他们去旅游了。
④ 抽象推进 —— go on / go ahead / go by / how it's going（超高頻！）
How's it going? 怎么样？（日常打招呼超级高频！）
Go ahead. 请便 / 去吧 / 继续说。（超级高频！让对方继续说/做）
What's going on? 发生什么事了？（超级高频疑问句）
Time goes by so fast. 时间过得真快。
Everything went well. 一切顺利。
Something went wrong. 出问题了。
The meeting didn't go as planned. 会议没有按计划进行。
I'll go first. 我先来。（物理/顺序上"先走"）
⑤ 固定短语 —— 必须要会的 GO 短语
Go for it! 去吧！/ 加油上！（超高頻鼓励话）
Go with the flow. 随大流 / 顺其自然。
It goes without saying. 这不用说也知道。（不言而喻）
There you go. 这就对了 / 给你。（超级高频口语短句）
Here we go again. 又来了 / 老样子。（又要重复了，无奈但高频）
Go over the report. 仔细检查/复习报告。
Go through the list. 逐一过一遍清单。
It doesn't go with your outfit. 它和你的衣服不搭。（go with = 搭配）
GO 的统一逻辑：GO 是英语里最基础的位移动词 —— "从A到B"。无论是物理上的去学校（go to school）、做活动（go swimming）、事情推进（how's it going），还是时间流逝（time goes by），本质都是从当前状态到另一个状态的移动。小品词决定了：往哪个方向去、到了没有、怎么去的。
GO 的过去式 = went（不规则）；过去分词 = gone（不在了）/ been（去过回来了）。
